# FactoryTalk Basic Lab Template

This application was generated from the template used in the **Introduction to FactoryTalk Optix Visualization Platform** FactoryTalk Optix 1.6 lab. It is identical to the base lab template and can be used in the same way as the application created during the initial lab steps.

---

## Application Files

In the main folder you will find the **FTOptixBasic.ACD** file containing the controller project (v36). This should be run on a physical PLC or in **FactoryTalk Echo**.  When adding a PLC in FactoryTalk Optix, make sure to set the correct controller path matching your configuration.

No other applications are required to complete all the exercises.

---

## Lab Materials

The **Lab Book** is not distributed together with this application. If you do not have access to it, please contact your **Rockwell Automation representative or authorized distributor**.

The entire lab is also available on our **onCourse** learning platform.

---

## Environment

The application was generated in **FactoryTalk Optix 1.6.2.36**.
